# TrieuBaoKhanh Expo UI (Product Fixed)

Đã sửa lại hẳn trang Product Detail:
- Căn lại khu quantity trừ / số / cộng cho gọn và đẹp hơn
- Giữ flow Home -> Product -> Explore -> Beverages
- Home vẫn có bottom bar cố định ở dưới, Explore tab chuyển sang Explore

## Chạy Expo Web
```bash
npm install
npx expo install --fix
npx expo install react-dom react-native-web @expo/metro-runtime
npx expo start --web -c
```
